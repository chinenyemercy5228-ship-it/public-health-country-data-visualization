# Public Health Country Data Visualization
 Project Overview
This project analyzes country-level economic and development indicators using Python, Pandas, and Matplotlib.

Visualizations
- GDP Category Distribution
- Nigeria vs Liberia GDP Trend
- Top 10 Countries by GDP
- GDP Per Capita vs Education Expenditure
- Population Density Distribution
- GDP Per Capita Across Continents
- Health vs Education Expenditure
- Population Trends
- Life Expectancy Analysis

 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
